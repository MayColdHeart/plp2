## Algoritmo de Pilha

### Uma abordagem comparativa entre Paradigmas Orientado à Objetos e Linguagem Funcional

Este repositório contém a demonstração prática do trabalho de avaliação de paradigmas de programação. O objetivo é comparar a linguagem de programação orientada à objetos com o paradigma de linguagem funcional através do algoritmo de pilha. Com este projeto será possível fazer uma análise de vantagens, desvantagens e limitações entre os paradigmas.

## Como testar as implementações

Abaixo estão os comandos necessários para compilar e rodar a demonstração de cada linguagem.

### Java (Paradigma Orientado a Objetos)

**Pré-requisito:** Ter o JDK (Java Development Kit) instalado.

1. Navegue até a pasta código fonte:

```cd Java/pilha/src/```

2. Compile o código do aplicativo principal (isso compilará as dependências automaticamente):

```javac App.java```

3. Execute o programa interativo:

```java App```

---

### 2. Clojure (Paradigma Funcional)

**Pré-requisito:**

1. Navegue até a pasta do Clojure:

```cd Clojure```

2. Copie o código do do arquivo.clj

```pilha.clj```

3. Para facilitar, execute o programa em um compilador online
